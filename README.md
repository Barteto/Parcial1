## 📋 Descripción

AutoBahia es una galería web estática que permite explorar una colección de fotografías de autos. El usuario puede encontrar fácilmente el auto que busca combinando un filtro por marca y un buscador por texto de forma simultánea, con retroalimentación inmediata gracias al contador de resultados.

---

## ✨ Funcionalidades

- **Filtro por marca** — Permite seleccionar una marca específica para mostrar solo los autos correspondientes.
- **Buscador por texto** — Busca autos en tiempo real mientras el usuario escribe.
- **Filtros simultáneos** — El filtro por marca y el buscador funcionan al mismo tiempo, combinando ambos criterios en una sola búsqueda.
- **Contador de resultados** — Muestra la cantidad de autos encontrados según los filtros aplicados.

## 🚀 Guía de instalación

No requiere instalación de dependencias. Al ser un proyecto estático, solo necesitás:

1. Clonar o descargar el repositorio:
   ```bash
   git clone https://github.com/tu-usuario/autobahia.git
   ```
2. Abrir el archivo `index.html` en tu navegador.

> También podés usar la extensión **Live Server** de VS Code para visualizarlo con recarga automática.
